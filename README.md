# Backend Practice Repository

This repository contains multiple backend practice projects built using **JavaScript**, **Node.js**, and **Express.js**.
The goal of this repository is to practice backend development concepts such as server creation, APIs, CRUD operations, routing, and JavaScript fundamentals.

# Project Descriptions

### Faulty Calculator

A JavaScript calculator program designed to practice debugging and logical problem solving.

### JS Strings

Practice examples demonstrating various string methods and operations in JavaScript.

### Business Name Generator

A small JavaScript project that generates random business names using word combinations.

### JS Arrays

Exercises covering array concepts such as:

* map()
* filter()
* reduce()
* push() / pop()

### Factorial Calculator

Implementation of factorial calculation using JavaScript loops and functions.

### JS Object Model

Basic practice of JavaScript object concepts and simple HTML interaction.

Clone the repository

```

```

```

```

Install dependencies

```
npm install
```

```
node index.js
```
